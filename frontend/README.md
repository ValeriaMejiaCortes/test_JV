# POSTS

Publica tu articulo

## Comenzando 🚀

debes realizar una copia de este proyecto dando clic en code y en el botón clone

### Pre-requisitos 📋

Para correr este repositorio. Necesitaras node and npm instalados globalmente en tu computador.

## Frontend

Instalación🔧:

```
npm install
```

Encender el servidor

```
npm start
```

### Backend🚀

Instalación🔧:

```
npm install
```

Encender el servidor

```
npm run dev
```
